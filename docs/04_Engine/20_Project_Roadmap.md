### Stage 1: ECS, Memory & Math Core (Completed)
* Contiguous Sparse Sets components registry.

* Shoelace area calculations.

### Stage 2: Physics, Kinetics & Collisions (Completed)

* Practice skill increments, promotions, shape affinities.
* Sentinel coordinate sharing memory loops.
