# Roadmap

This document outlines the epics list and technical milestones for development.

---

## 1. Project Epics Mappings

| Epic ID | Title | Description |
|---------|-------|-------------|
| **EPIC-001** | Core Engine Foundation | Types, assertions, async logger targets |
| **EPIC-002** | Platform & Build System | SDL3 window bootstrap, CMake configurations |
| **EPIC-003** | Memory & Containers | Arena and Pool Allocators, Sparse Set ECS |
| **EPIC-004** | Mathematics & Geometry | Custom Vector2, polygon Shoelace calculations |
| **EPIC-005** | ECS & Job System | Multi-threaded jobs scheduler, CommandBuffers |
| **EPIC-006** | Simulation Kernel | Clock timings, unified ticks scheduler |
| **EPIC-007** | Physics Engine | Euler-Cromer kinematics, SAT collision resolvers |
| **EPIC-008** | Rendering Engine | Render Graph draw abstractions, shaders |
| **EPIC-009** | AI & Cognition | Needs, memory tables, Utility AI evaluations |
| **EPIC-010** | Biology & Evolution | Genetics crossover and Gaussian mutations |
| **EPIC-011** | Civilization Systems | Storage depots, technology trees, price formulas |
| **EPIC-012** | Editor & Tooling | ImGui inspectors, spatial grid overlays |
| **EPIC-013** | Networking & Replay | Lockstep packets synchronization, rollback |
| **EPIC-014** | Optimization & Profiling | L1/L2 cache tuning, LOD tick decoupling |
| **EPIC-015** | First Playable Prototype | Spawners, HUD control setups |

---

7. Execution Model
8. API & Interface Design
9. Repository & Build Infrastructure
10. Testing Strategy
11. Implementation
12. Integration
13. Verification & Validation
