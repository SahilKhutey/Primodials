# 18_Tutorial_Workflow

This document specifies the interactive tutorial stages, guided instructions, and learning objectives.

---

## 1. Tutorial Structure
The game introduces players to the simulation through three guided scenarios:

```
[Tutorial 1: Needs & Survival] ──→ [Tutorial 2: Sentinel Command] ──→ [Tutorial 3: Carrier Logistics]
```

---

## 2. Guided Scenarios

### Tutorial 1: Needs & Survival
* **Goal:** Learn how shapes manage hunger and discover resource food nodes.
* **Instruction:** The player drops resource attractors, guiding starving triangles toward food coordinates.

### Tutorial 2: Sentinel Command
* **Goal:** Understand command fields and policies.
* **Instruction:** The player constructs a Sentinel node, select the "Harvest" policy to coordinate gatherer paths.

### Tutorial 3: Carrier Logistics
* **Goal:** Optimize transport between distant depots.
* **Instruction:** The player constructs a trade highway between a farm and a workshop, assigning carrier squares to balance stock.
